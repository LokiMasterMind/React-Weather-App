#React Weather App

To use this repo
### Clone This Repo.
### Change API key in **api\openWeatherMap.jsx** folder. if you don't have [Click Here!](http://openweathermap.org/) and get your API key.
```
const API_KEY="your-api-key";
```
## Steps to run :
Terminal 1: 
```
   $ npm install
   $ wepack -w
```   
Terminal 2: 
```
   $ node server.js
```
Open in Browser:

[http://localhost:3000/](http://localhost:3000/)

### Enjoy :-)
